QL Vật Tư Công Trình - Demo (with lightweight login)
=======================================================

Quick start:
1. Extract the zip.
2. Install dependencies:
   npm install
3. Run dev:
   npm run dev
4. Open http://localhost:5173

Deploy to Vercel:
- Push to GitHub, then import repo in Vercel.
- Build command: npm run build
- Output directory: dist

Notes:
- Lightweight login stores name/email in localStorage (no password) for demo.
- All users have equal permissions in this demo.
